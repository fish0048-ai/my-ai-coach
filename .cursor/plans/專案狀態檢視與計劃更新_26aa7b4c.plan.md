### 2. UI 優化（可選）

**目標**：不只視覺優化，而是**全頁面風格統一**＋**人性化設計**，提升整體使用體驗。

#### 2.1 風格統一

- 定義 Design Token（色彩、圓角、間距、字級）於 `tailwind.config.js`、`index.css`
- **全頁面**套用一致風格：Dashboard、Calendar、Nutrition、Profile、KnowledgeBase、Trend、Gear、AI Coach 等
- 風格方向：Modern Fitness Dark（深色為主、數據可讀）＋ Glassmorphism AI Panel（AI 教練區塊磨砂玻璃）

#### 2.2 人性化設計

- **導航與回饋**：明確的麵包屑、返回按鈕、載入狀態、成功／錯誤提示
- **可讀性**：適當對比、字級階層、重要資訊突出
- **操作友善**：按鈕大小適合觸控、表單欄位有 placeholder 與驗證提示
- **無障礙**：考量鍵盤操作、色彩對比、語意化標籤

#### 2.3 建議修改範圍


| 類別               | 檔案                                                                                            |
| ---------------- | --------------------------------------------------------------------------------------------- |
| **Design Token** | tailwind.config.js、src/index.css                                                              |
| **Layout**       | MainLayout.jsx（導航、AI Coach 區塊）                                                                |
| **核心 View**      | DashboardView、CalendarView、NutritionView、Profile、KnowledgeBaseView、TrendAnalysisView、GearView |
| **共用元件**         | StatCard、CoachChat、ErrorToast 等                                                               |


**建議順序**：先定義 Token → 調整 MainLayout → 逐頁統一風格 → 人性化細節優化
